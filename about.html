The developer of KATSU is not responsible for the contents that can be displayed of any of the Modules. KATSU by itself can't do anything besides load and intepret Modules. The code of the modules is in JSON format.

Modules can be created and imported by anyone. The user is responsible of the modules that he decide to load on KATSU.

If you think a module is displaying content or data that is copyrighted we recommend you to go to the real provider, which can be found in the module code : Base Url Section

If you are the owner of any of the content that is displayed on a Module and can be found in the Module library. We invite you to provided proofs that certificate that you have it's rights of it and if enough information is provided it will be blacklisted from the module instantly.

Modules are just pieces of instructions that KATSU will follow to display data on it. Modules dont contain any type of data. Modules tell the app to go to public websites and scrape the data to display it to the user.

KATSU doesn't host or provide any type of content to the user, the Modules that the user decide to load are the ones that will collect the data from public websites and display it. The user decides the modules that he wants to load on KATSU, the developer of KATSU is not responsible for the modules that the user decide to load on KATSU.

To report a module go to the Modules Library on KATSU and slide it to the left then select report. You can report it as well, sending an email to appkatsu@gmail.com with the module id and the information in the email.
